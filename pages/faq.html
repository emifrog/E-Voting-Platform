❓ Questions Fréquentes

Est-ce vraiment gratuit ? Quels sont les coûts cachés ?

✅ 100% gratuit, open source (licence MIT). Seuls coûts : votre hébergement (~5-20€/mois selon trafic) et optionnellement un nom de domaine (~10€/an).

Dois-je être développeur pour l'installer ?

Non ! Notre guide d'installation est conçu pour les non-techniques. Si vous savez utiliser un terminal, vous pouvez le déployer en 5 minutes. Sinon, la communauté peut vous aider.

Comment se compare E-Voting à Voteer, Balotilo, etc. ?

Principal avantage : open source + gratuit. Vous possédez vos données, contrôlez votre infrastructure. Les concurrents facturent 2-5€ par votant, soit 500-2000€ par AG de 100-400 personnes.

Puis-je l'utiliser pour des élections légales/officielles ?

Oui, si votre juridiction l'autorise. E-Voting respecte les standards (chiffrement AES-256, audit trail, 2FA). Consultez un avocat pour confirmer la conformité avec vos règles locales.

Quelle est la limite de votants ?

Testé jusqu'à 30,000 votants. Avec optimisation (Redis, load balancing), peut supporter 100,000+. Pour <1000 votants, un serveur basique suffit.

Offrez-vous un support payant pour entreprises ?

Actuellement : support communautaire gratuit (GitHub Issues, Discord). Support premium/consulting à venir. Contactez-nous pour besoins spécifiques.
